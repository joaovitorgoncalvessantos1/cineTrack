# 🎬 CineTrack

## 📌 Visão Geral

O CineTrack é uma aplicação web desenvolvida em React que permite aos usuários explorar filmes populares, visualizar informações detalhadas e criar avaliações pessoais salvas diretamente no navegador.

O projeto simula uma experiência de produto real, semelhante ao IMDb ou Letterboxd, com foco em consumo de API, gerenciamento de estado e interface limpa.

## 🎯 Funcionalidades

- 🔍 Listagem de filmes populares via API externa
- 🎬 Visualização de detalhes do filme (título, sinopse, nota, ano, gêneros)
- ⭐ Criação de avaliações personalizadas
- 🧠 Adição de emoções e recomendação (sim/não)
- 💾 Salvamento de avaliações no LocalStorage
- 📄 Visualização de todas as avaliações salvas
- 🧭 Navegação entre páginas (home, detalhes, formulário e avaliações)

## ⚙️ Como funciona

1. Os filmes são carregados a partir de uma API externa
2. A página inicial exibe filmes populares em formato de grade
3. Ao clicar em um filme, o usuário acessa os detalhes
4. O usuário pode criar uma avaliação contendo:
   - nota
   - texto da avaliação
   - recomendação (sim/não)
   - emoções relacionadas ao filme
5. As avaliações são salvas no LocalStorage
6. O usuário pode visualizar todas as avaliações posteriormente

---

## 💾 Persistência de Dados

As avaliações são armazenadas no LocalStorage do navegador, permitindo que os dados permaneçam salvos mesmo após recarregar a página, sem necessidade de backend.
